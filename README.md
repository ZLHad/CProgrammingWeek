# 程序设计周-随手记账
## 准备

安装node.js

### 运行

安装依赖：  
```bash
npm install
```
启动服务React：  
``` bash
npm start
```
启动electron调试模式：  
``` bash
npm run electron-dev
```
### 提交
拉取代码：  
```bash
git pull origin master
```
提交代码：  
``` bash
git add .
git commit -m "描述"
git push origin master
```
